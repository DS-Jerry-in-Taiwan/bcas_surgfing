# Phase 1.1 - 資料探索與管道建置

**階段**: Phase 1.1 (Data Discovery & Pipeline)
**專案**: Project Gamma Surf (CBAS 量化交易系統)

## 🎯 開發目標
本階段不急於建立正式資料庫，而是透過「原型爬蟲」獲取原始數據，進行探索性分析 (EDA)，以確保後續 Schema 設計的正確性。

### 核心產出
1.  **原型爬蟲腳本**: 能抓取 TWSE (現股) 與 TPEx (CB) 近期日線數據的 Python Script。
2.  **原始數據樣板**: 下載並保存為 JSON/CSV 格式的 Raw Data。
3.  **數據分析報告**: 包含欄位定義、缺值狀況、異常值與主鍵 (Primary Key) 建議。
4.  **Schema 草案**: 基於分析結果的 SQL Table 定義建議。

### 驗收標準
- [ ] 爬蟲能成功抓取指定股票 (如 2330) 與 CB (如 52694) 的日線資料。
- [ ] 能正確處理民國年 (e.g., 113/01/01) 轉西元年。
- [ ] 產出的 CSV 檔案無亂碼，數值欄位已移除逗號。
- [ ] 數據分析報告確認了「轉換價」與「除權息」的資料特徵。
