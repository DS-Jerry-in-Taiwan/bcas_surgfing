# Phase 1.1 - Agent 角色職責

## 🏗️ @INFRA (環境工程師)
- **職責**: 準備 Python 虛擬環境與資料夾結構。
- **重點**: 確保 `data/` 目錄結構清晰，且 `.gitignore` 設定正確 (忽略 CSV/JSON)。

## 📐 @ARCH (架構師)
- **職責**: 設計爬蟲的擴充性架構。
- **重點**: 
    - 定義 Crawler 的標準介面，確保未來新增 XAU/USD 時不用重寫邏輯。
    - 規劃 Rate Limiting 機制，避免被證交所封 IP。

## 💻 @CODER (開發工程師)
- **職責**: 撰寫實際的 Python 爬蟲代碼。
- **重點**: 
    - 處理台灣股市特有的資料格式 (民國年、逗號分隔數值)。
    - 實作錯誤重試 (Retry) 機制。

## 🧪 @ANALYST (測試與數據分析師)
- **職責**: 驗證抓下來的資料品質。
- **重點**: 
    - 檢查 CB 資料中的「成交量」是否為 0 或 Null。
    - 確認「轉換價」欄位是否存在於每日行情表中，還是需要另外爬取。
