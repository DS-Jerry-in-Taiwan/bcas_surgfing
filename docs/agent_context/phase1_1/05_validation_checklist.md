# Phase 1.1 - 驗證清單

## 功能驗證
- [ ] `TwseCrawler` 能成功抓取指定月份的 JSON 數據。
- [ ] `TpexCbCrawler` 能成功抓取指定日期的 CSV/JSON 數據。
- [ ] 民國年轉換函式測試通過 (e.g., 112/02/29 -> 2023-02-29)。

## 數據品質驗證
- [ ] 數值欄位 (Open/High/Low/Close) 均為 float/int 型態，無字串 '1,000'。
- [ ] 日期格式統一為 `YYYY-MM-DD`。
- [ ] 檔案儲存路徑符合規範 `data/raw/{type}/{date}.csv`。

## 效能與風控
- [ ] 爬蟲請求間隔是否有隨機延遲 (避免固定頻率)。
- [ ] 當遇到 403/429 錯誤時，是否有重試機制？
